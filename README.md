## Public positioning

The site presents two active investment theses:

- **GAO — Governed Agentic Ops:** runtime control, identity, permissions, intervention, evaluation, and audit evidence for agentic systems.
- **VSRAI — Vertical System-of-Record AI:** authoritative write-back, workflow ownership, defensible data rights, and repeatable implementation in regulated verticals.

SSI v4.0 is the evidence-led investment method behind the work. The public site explains the operating contract — Fact / Inference / Uncertainty, one primary thesis, separated risk overlays, falsifiers, and dated next verification — without publishing private company scores or decisions.
